Explorando Objetos em JavaScript

Aprenda a trabalhar com objetos, uma estrutura fundamental para organizar dados.

O que é um Objeto?

Em JavaScript, um objeto é uma coleção de pares chave-valor, onde as chaves são strings e os valores podem ser de qualquer tipo de dado.

Exemplo de declaração de um objeto:

                
                    let pessoa = {
                        nome: 'João',
                        idade: 30,
                        cidade: 'São Paulo'
                    };

Acesso a Propriedades

As propriedades de um objeto podem ser acessadas usando a notação de ponto ou colchetes.

Exemplo de acesso a propriedades:

                
                    console.log(pessoa.nome);  // Saída: João
                    console.log(pessoa['idade']);  // Saída: 30

Métodos de Objeto

Além de armazenar dados, os objetos em JavaScript podem conter funções, chamadas de métodos.

Exemplo de método em um objeto:

                
                    let pessoa = {
                        nome: 'Maria',
                        saudacao: function() {
                            console.log('Olá, meu nome é ' + this.nome + '!');
                        }
                    };

                    pessoa.saudacao();  // Saída: Olá, meu nome é Maria!

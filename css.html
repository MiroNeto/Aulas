Introdução ao CSS

O CSS (Cascading Style Sheets) é uma linguagem de estilo usada para controlar a apresentação visual de uma página HTML. Permite definir o layout, cores e estilos para os elementos HTML.

O CSS é fundamental para criar sites atraentes e bem organizados. Vamos explorar os principais conceitos e propriedades do CSS neste guia.

Seletores CSS

Os seletores CSS são padrões que são usados para selecionar e estilizar os elementos HTML. Eles podem ser simples, como a seleção de uma tag, ou mais complexos, como seleções baseadas em classes ou IDs.

Exemplo de seleção por tag:


p {
    color: blue;
}

Cores em CSS

As cores podem ser especificadas de várias maneiras em CSS, incluindo nomes de cores, códigos hexadecimais, RGB, RGBA e muito mais. As cores são essenciais para a aparência de um site.

Exemplos:


body {
    background-color: white;
}

h1 {
    color: #FF0000;
}

Layouts com CSS

O CSS permite criar diferentes layouts para as páginas web. Você pode usar flexbox, grids, posicionamento e outros recursos para organizar o conteúdo de maneira eficaz.

Exemplo de uso de flexbox:


.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

Flexbox em CSS

Flexbox é um layout unidimensional que permite organizar itens dentro de um contêiner de maneira mais eficiente. É amplamente utilizado para criar layouts flexíveis e responsivos.

Exemplo de uso de flexbox:


.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

Grid em CSS

O Grid Layout é uma ferramenta bidimensional que permite a criação de layouts complexos e flexíveis. É uma ótima opção para alinhar elementos em linhas e colunas.

Exemplo de uso de grid:


.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-gap: 10px;
}

Transições e Animações

Transições e animações são formas de adicionar movimento e interatividade aos elementos HTML usando CSS. Podem ser aplicadas a propriedades como cor, tamanho e posição.

Exemplo de transição suave:


.button {
    transition: background-color 0.3s ease;
}

.button:hover {
    background-color: #555;
}

Responsividade e Media Queries

É fundamental criar layouts que se adaptem a diferentes dispositivos e tamanhos de tela. As Media Queries permitem aplicar estilos com base nas características do dispositivo.

Exemplo de uso de Media Query:


@media (max-width: 600px) {
    body {
        font-size: 14px;
    }
}

Box Model

O Box Model é uma representação visual de como os elementos HTML são renderizados no navegador. Ele inclui propriedades como margin, border, padding e content.

Exemplo de uso do Box Model:


.box {
    width: 200px;
    padding: 20px;
    border: 2px solid #333;
    margin: 10px;
}

Transformações CSS

As transformações CSS permitem alterar a aparência de um elemento de uma maneira 2D ou 3D. Você pode aplicar rotações, escalas, translações e inclinações.

Exemplo de transformações 2D:


.element {
    transform: rotate(45deg);
}

Filtros CSS

Os filtros CSS permitem aplicar efeitos visuais em elementos HTML, como desfoque, saturação, brilho, entre outros. São úteis para ajustar a estética das imagens.

Exemplo de aplicação de filtro:


.image {
    filter: grayscale(50%);
}

Posicionamento CSS

O posicionamento em CSS é crucial para controlar a localização dos elementos em uma página web. Você pode usar position, top, bottom, left e right para isso.

Exemplo de posicionamento absoluto:


.element {
    position: absolute;
    top: 20px;
    left: 30px;
}

Tipografia e Fontes

A escolha adequada da tipografia e das fontes é fundamental para a legibilidade e o apelo visual de um site. O CSS oferece várias propriedades para estilizar o texto.

Exemplo de estilização de fonte:


body {
    font-family: 'Arial', sans-serif;
    font-size: 16px;
}

Gradientes CSS

Os gradientes CSS permitem criar transições suaves entre duas ou mais cores. Podem ser lineares ou radiais, adicionando profundidade e estilo aos elementos.

Exemplo de gradiente linear:


.element {
    background: linear-gradient(to right, red, yellow);
}

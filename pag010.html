Objetos em JavaScript

Aprenda sobre objetos, uma estrutura fundamental em JavaScript.

O que são objetos?

Em JavaScript, um objeto é uma coleção de propriedades, onde cada propriedade possui um nome e um valor.

Exemplo de objeto:

                
                    let pessoa = {
                        nome: 'João',
                        idade: 30,
                        cidade: 'São Paulo'
                    };

Acesso a propriedades

Você pode acessar as propriedades de um objeto usando a notação de ponto ou colchetes.

Exemplos:

                
                    console.log(pessoa.nome);  // Acesso usando ponto
                    console.log(pessoa['idade']);  // Acesso usando colchetes

Adição de propriedades

Você pode adicionar novas propriedades a um objeto a qualquer momento.

Exemplo:

                
                    pessoa.profissao = 'Programador';

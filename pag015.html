Explorando Arrays em JavaScript

Descubra como trabalhar com coleções de dados utilizando arrays.

O que é um Array?

Um array em JavaScript é uma estrutura que permite armazenar uma coleção de elementos.

Exemplo de declaração de um array:

                
                    const frutas = ['Maçã', 'Banana', 'Laranja'];

Acesso aos Elementos

Os elementos de um array podem ser acessados através de índices numéricos.

Exemplo de acesso a elementos de um array:

                
                    console.log(frutas[0]);  // Saída: Maçã
                    console.log(frutas[1]);  // Saída: Banana

Modificação de Arrays

Os arrays podem ser modificados através de operações como adição, remoção e modificação de elementos.

Exemplo de adição de um elemento ao final do array:

                
                    frutas.push('Abacaxi');
                    console.log(frutas);  // Saída: ['Maçã', 'Banana', 'Laranja', 'Abacaxi']

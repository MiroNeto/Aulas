Funções em JavaScript

Aprenda a criar e utilizar funções para organizar seu código.

O que é uma função?

Uma função é um bloco de código que pode ser chamado e executado. Ela pode receber parâmetros e retornar um valor.

Exemplo de declaração de função:

                
                    function saudacao(nome) {
                        return 'Olá, ' + nome + '!';
                    }

                    let mensagem = saudacao('João');
                    console.log(mensagem);  // Saída: Olá, João!

Parâmetros e Argumentos

As funções podem receber parâmetros, que são valores passados quando a função é chamada. Esses valores são chamados de argumentos.

Exemplo de função com parâmetros:

                
                    function soma(a, b) {
                        return a + b;
                    }

                    let resultado = soma(10, 5);
                    console.log(resultado);  // Saída: 15

Retorno de Função

Uma função pode retornar um valor usando a palavra-chave return. Isso finaliza a execução da função e retorna o valor especificado.

Exemplo de retorno de função:

                
                    function quadrado(numero) {
                        return numero * numero;
                    }

                    let quadradoDe5 = quadrado(5);
                    console.log(quadradoDe5);  // Saída: 25

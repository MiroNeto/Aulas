Estrutura Básica de um Documento HTML5

O HTML5 introduziu uma estrutura mais simples para criar páginas web. Veja um exemplo básico abaixo:


<!DOCTYPE html>
<html>
<head>
    <title>Título da Página</title>
</head>
<body>
    Conteúdo da Página
</body>
</html>

Neste exemplo:

  • <!DOCTYPE html>: Define que este é um documento HTML5.
  • <html>: Contém todo o conteúdo da página.
  • <head>: Contém informações sobre o documento, como o título.
  • <title>: Define o título da página exibido na barra de título do navegador.
  • <body>: Contém o conteúdo visível da página.

Este é o esqueleto básico que você deve usar ao iniciar uma nova página HTML5.

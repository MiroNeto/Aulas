Tipos de Dados em JavaScript

Entenda os diferentes tipos de dados que podem ser usados em JavaScript.

Números

Em JavaScript, os números podem ser inteiros ou decimais.

Exemplo:

                
                    let inteiro = 10;
                    let decimal = 3.14;

Strings

Strings representam sequências de caracteres e devem ser colocadas entre aspas simples, duplas ou backticks.

Exemplo:

                
                    let nome = 'Alice';
                    let frase = "Olá, mundo!";
                    let template = \`Meu nome é \${nome}\`;

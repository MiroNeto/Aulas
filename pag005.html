Variáveis em JavaScript

Entenda o conceito de variáveis em JavaScript e como utilizá-las.

Declaração de Variáveis

Em JavaScript, usamos var, let e const para declarar variáveis.

Exemplo:

                
                    var nome = 'Alice'; // Declaração usando var
                    let idade = 25; // Declaração usando let
                    const PI = 3.14; // Declaração usando const

Escopo de Variáveis

O escopo de uma variável determina onde ela pode ser acessada dentro do código.

Exemplo:

                
                    let x = 10; // Variável global

                    function teste() {
                        let y = 5; // Variável local dentro da função
                        console.log(x); // Pode acessar x
                        console.log(y); // Pode acessar y
                    }

                    console.log(x); // Pode acessar x
                    console.log(y); // Erro: y não está definido fora da função

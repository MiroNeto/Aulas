Explorando Objetos em JavaScript

Conheça os objetos, estruturas fundamentais que armazenam dados e funcionalidades.

O que é um Objeto?

Em JavaScript, um objeto é uma estrutura que armazena dados e funcionalidades relacionadas.

Exemplo de um objeto representando uma pessoa:

                
                    const pessoa = {
                        nome: 'João',
                        idade: 30,
                        saudacao: function() {
                            return 'Olá, meu nome é ' + this.nome + ' e tenho ' + this.idade + ' anos.';
                        }
                    };

Propriedades de um Objeto

Os objetos contêm propriedades que representam características ou informações associadas a eles.

Exemplo de acesso a propriedades de um objeto:

                
                    console.log(pessoa.nome);  // Saída: João
                    console.log(pessoa.idade);  // Saída: 30

Métodos de um Objeto

Métodos são funções associadas a objetos e permitem que os objetos realizem ações.

Exemplo de chamada de método de um objeto:

                
                    console.log(pessoa.saudacao());  // Saída: Olá, meu nome é João e tenho 30 anos.

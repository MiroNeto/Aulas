Estruturas de Controle em JavaScript

Aprenda a controlar o fluxo do seu programa com estruturas condicionais e de repetição em JavaScript.

Estruturas Condicionais

As estruturas condicionais permitem que o programa tome decisões baseadas em condições.

Exemplo de estrutura condicional if-else:

                
                    let idade = 20;

                    if (idade >= 18) {
                        console.log('Você é maior de idade.');
                    } else {
                        console.log('Você é menor de idade.');
                    }

Loops (Laços de Repetição)

Os loops permitem que um bloco de código seja repetido enquanto uma condição for verdadeira.

Exemplo de loop for:

                
                    for (let i = 0; i < 5; i++) {
                        console.log('Número:', i);
                    }

Estrutura switch

O switch permite executar diferentes blocos de código baseados no valor de uma variável.

Exemplo de switch:

                
                    let diaSemana = 'segunda';

                    switch (diaSemana) {
                        case 'segunda':
                            console.log('Hoje é segunda-feira.');
                            break;
                        case 'terca':
                            console.log('Hoje é terça-feira.');
                            break;
                        default:
                            console.log('Dia não reconhecido.');
                    }

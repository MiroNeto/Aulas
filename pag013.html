Entendendo Funções em JavaScript

Aprenda sobre funções, uma parte fundamental da programação em JavaScript.

O que é uma Função?

Uma função é um bloco de código reutilizável que realiza uma tarefa específica.

Exemplo de uma função em JavaScript:

                
                    function saudacao(nome) {
                        return 'Olá, ' + nome + '!';
                    }

Chamando uma Função

Para usar uma função, você precisa chamá-la pelo seu nome e fornecer os argumentos necessários, se houver.

Exemplo de chamada de função:

                
                    const mensagem = saudacao('João');
                    console.log(mensagem);  // Saída: Olá, João!

Parâmetros de Função

Parâmetros são valores que uma função aceita quando é chamada.

Exemplo de função com parâmetros:

                
                    function soma(a, b) {
                        return a + b;
                    }

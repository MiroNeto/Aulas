Operadores em JavaScript

Entenda como usar operadores em JavaScript para realizar operações em variáveis e valores.

Operadores Aritméticos

Os operadores aritméticos permitem fazer operações matemáticas básicas.

Exemplo:

                
                    let a = 10;
                    let b = 5;

                    let soma = a + b;
                    let subtracao = a - b;
                    let multiplicacao = a * b;
                    let divisao = a / b;

Operadores Relacionais

Os operadores relacionais permitem comparar valores.

Exemplo:

                
                    let x = 10;
                    let y = 5;

                    let maiorQue = x > y;
                    let menorQue = x < y;
                    let igualA = x === y;

Variáveis e Constantes em JavaScript

Entenda como utilizar variáveis e constantes em JavaScript.

Variáveis

Em JavaScript, uma variável é um contêiner para armazenar dados. Você pode usar variáveis para armazenar várias informações.

Exemplo:

                
                    let idade = 30; // Declaração de variável
                    console.log(idade); // Acesso à variável

Constantes

Em JavaScript, uma constante é um tipo de variável cujo valor não pode ser alterado após a atribuição inicial.

Exemplo:

                
                    const PI = 3.14; // Declaração de constante
                    console.log(PI); // Acesso à constante

Trabalhando com Arrays em JavaScript

Descubra como usar e manipular arrays para armazenar e organizar dados em JavaScript.

O que é um Array?

Um array é uma estrutura de dados que pode armazenar mais de um valor ou objeto de uma vez sob um único nome.

Exemplo de array:

                
                    const frutas = ['Maçã', 'Banana', 'Laranja', 'Morango'];

Acesso aos elementos

Os elementos de um array podem ser acessados através de um índice, que começa em 0 para o primeiro elemento.

Exemplo:

                
                    const frutas = ['Maçã', 'Banana', 'Laranja', 'Morango'];
                    console.log(frutas[0]);  // Acesso ao primeiro elemento (Maçã)

Modificação de um Array

Arrays podem ser modificados adicionando, removendo ou alterando elementos.

Exemplos:

                
                    const frutas = ['Maçã', 'Banana', 'Laranja', 'Morango'];
                    frutas.push('Uva');  // Adiciona 'Uva' no final do array
                    frutas.pop();  // Remove o último elemento do array
                    frutas[1] = 'Pera';  // Altera o segundo elemento para 'Pera'

Guia Completo de HTML5

Estrutura Básica de um Documento HTML5


<!DOCTYPE html>
<html>
<head>
    <title>Título da Página</title>
</head>
<body>
    Conteúdo da Página
</body>
</html>

Principais Elementos de Texto

Títulos


<h1>Título Principal</h1>
<h2>Subtítulo</h2>
...
<h6>Título de menor importância</h6>

Parágrafos


<p>Este é um parágrafo de texto.</p>

Estilos de Texto


<strong>Texto em negrito</strong>
<em>Texto em itálico</em>

Listas

Listas Não Ordenadas


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Listas Ordenadas


<ol>
    <li>Item 1</li>
    <li>Item 2</li>
</ol>

Links

Links para Email


<a href="mailto:exemplo@email.com">Enviar Email</a>

Links Externos


<a href="https://www.example.com" target="_blank">Link Externo</a>

Imagens

Inserindo Imagens


<img src="caminho/da/imagem.jpg" alt="Descrição da Imagem">

Formulários

Campos de Entrada


<input type="text" placeholder="Nome">
<input type="password" placeholder="Senha">
<input type="email" placeholder="Email">
<input type="number" placeholder="Número">

Botões


<button>Enviar</button>
<input type="submit" value="Enviar">

Dropdowns com <select>


<select>
    <option value="opcao1">Opção 1</option>
    <option value="opcao2">Opção 2</option>
    <option value="opcao3">Opção 3</option>
</select>

Envio de Arquivos


<input type="file">

Tabelas

<table>
    <tr>
        <th>Cabeçalho 1</th>
        <th>Cabeçalho 2</th>
    </tr>
    <tr>
        <td>Dado 1</td>

    </tr>
</table>

Inserindo Vídeo


<video controls>
    <source src="caminho/do/video.mp4" type="video/mp4">

</video>

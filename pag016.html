Explorando Funções em JavaScript

Aprenda a criar e utilizar funções para organizar e reutilizar seu código.

O que é uma Função?

Uma função em JavaScript é um bloco de código reutilizável, projetado para realizar uma tarefa específica.

Exemplo de declaração de uma função:

                
                    function saudacao() {
                        console.log('Olá, seja bem-vindo!');
                    }

Chamada de Funções

Para usar uma função, você precisa chamá-la pelo nome, seguido por parênteses.

Exemplo de chamada de uma função:

                
                    saudacao();  // Saída: Olá, seja bem-vindo!

Parâmetros e Argumentos

Funções podem receber informações, chamadas de parâmetros, que são passadas durante a chamada da função.

Exemplo de função com parâmetros:

                
                    function saudar(nome) {
                        console.log('Olá, ' + nome + '!');
                    }

                    saudar('Alice');  // Saída: Olá, Alice!

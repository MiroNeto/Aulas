Funções em JavaScript

Entenda como usar funções para organizar e reutilizar seu código em JavaScript.

Declaração de Funções

Em JavaScript, você pode declarar funções usando a palavra-chave function.

Exemplo:

                
                    function saudacao() {
                        console.log('Olá, seja bem-vindo!');
                    }

                    // Chamando a função
                    saudacao();

Parâmetros de Funções

Você pode passar parâmetros para funções para torná-las mais flexíveis e reutilizáveis.

Exemplo:

                
                    function saudar(nome) {
                        console.log('Olá, ' + nome + '!');
                    }

                    // Chamando a função com um parâmetro
                    saudar('João');

Retorno de Funções

As funções podem retornar valores usando a palavra-chave return.

Exemplo:

                
                    function multiplicar(a, b) {
                        return a * b;
                    }

                    let resultado = multiplicar(2, 3);
                    console.log('O resultado da multiplicação é: ' + resultado);

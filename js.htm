História do JavaScript

O JavaScript foi criado em 1995 por Brendan Eich enquanto trabalhava na Netscape Communications Corporation. Inicialmente, era chamado de Mocha, depois renomeado para LiveScript e, finalmente, para JavaScript.

Hoje, o JavaScript é uma das linguagens de programação mais populares do mundo, sendo amplamente utilizada para o desenvolvimento web.

Tabelas em HTML5

As tabelas são usadas para exibir dados em formato tabular. Veja como criar uma tabela simples em HTML5:

Criando uma Tabela Simples


<table>
    <tr>
        <th>Cabeçalho 1</th>
        <th>Cabeçalho 2</th>
    </tr>
    <tr>
        <td>Dado 1</td>
        <td>Dado 2</td>
    </tr>
</table>

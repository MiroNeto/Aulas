Tipos de Dados em JavaScript

Conheça os principais tipos de dados utilizados em JavaScript.

Tipos Básicos

JavaScript possui os tipos de dados básicos:

  • Number
  • String
  • Boolean
  • Null
  • Undefined

Exemplo:

                
                    let idade = 25; // Tipo Number
                    let nome = 'João'; // Tipo String
                    let isAluno = true; // Tipo Boolean
                    let preco = null; // Tipo Null
                    let indefinido; // Tipo Undefined

Tipos Complexos

JavaScript também possui tipos de dados complexos:

  • Object
  • Array
  • Function

Exemplo:

                
                    let pessoa = {
                        nome: 'Maria',
                        idade: 30
                    }; // Tipo Object

                    let numeros = [1, 2, 3, 4]; // Tipo Array

                    function somar(a, b) {
                        return a + b;
                    } // Tipo Function

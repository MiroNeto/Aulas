Funções em JavaScript

Descubra como criar e usar funções, blocos de código reutilizáveis em JavaScript.

O que são funções?

Em JavaScript, uma função é um bloco de código reutilizável que realiza uma tarefa específica. Funções permitem modularizar o código e torná-lo mais organizado e legível.

Exemplo de função:

                
                    function saudacao() {
                        console.log('Olá! Bem-vindo ao nosso site.');
                    }

Chamada de Função

Para usar uma função, você precisa chamá-la. Isso é feito colocando parênteses após o nome da função.

Exemplo:

                
                    saudacao();  // Chamando a função saudacao

Parâmetros e Argumentos

Uma função pode receber parâmetros, que são valores passados durante a chamada da função. Esses valores são chamados de argumentos.

Exemplo:

                
                    function saudacao(nome) {
                        console.log('Olá, ' + nome + '! Bem-vindo ao nosso site.');
                    }
                    saudacao('João');  // Chamando a função com um argumento

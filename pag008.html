Condicionais e Estruturas de Controle em JavaScript

Entenda como usar condicionais e estruturas de controle para controlar o fluxo de execução em JavaScript.

if-else

A estrutura de controle if-else permite executar diferentes blocos de código dependendo de uma condição.

Exemplo:

                
                    let idade = 18;

                    if (idade >= 18) {
                        console.log('Pode dirigir.');
                    } else {
                        console.log('Não pode dirigir.');
                    }

switch

O switch permite executar diferentes blocos de código dependendo do valor de uma variável.

Exemplo:

                
                    let diaSemana = 'segunda';

                    switch (diaSemana) {
                        case 'segunda':
                            console.log('Hoje é segunda-feira.');
                            break;
                        case 'terca':
                            console.log('Hoje é terça-feira.');
                            break;
                        default:
                            console.log('Dia não encontrado.');
                    }

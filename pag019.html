Funções em JavaScript

Aprenda sobre a criação e utilização de funções para organizar e reutilizar código em JavaScript.

O que é uma função?

Uma função é um bloco de código que pode ser chamado e executado em diferentes partes de um programa. Ela pode aceitar parâmetros e retornar um valor.

Exemplo de uma função simples:

                
                    function saudacao() {
                        console.log('Olá! Bem-vindo.');
                    }

                    saudacao();  // Chama a função para exibir a saudação

Parâmetros de uma função

As funções podem aceitar parâmetros, que são valores que a função espera para executar suas ações.

Exemplo de uma função com parâmetros:

                
                    function saudacao(nome) {
                        console.log('Olá, ' + nome + '! Bem-vindo.');
                    }

                    saudacao('João');  // Chama a função passando 'João' como parâmetro

Retorno de uma função

Uma função pode retornar um valor utilizando a palavra-chave return. Esse valor pode ser utilizado em outras partes do programa.

Exemplo de uma função que retorna um valor:

                
                    function soma(a, b) {
                        return a + b;
                    }

                    let resultado = soma(5, 3);
                    console.log('A soma é:', resultado);
